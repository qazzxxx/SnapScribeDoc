# 🔧 安装指南

## 系统要求

- **浏览器**：Chrome 88+ 或其他基于 Chromium 的浏览器
- **操作系统**：Windows 10+、macOS 10.14+、Linux
- **权限**：需要访问活动标签页和截图权限
